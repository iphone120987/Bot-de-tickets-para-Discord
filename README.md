Projeto do bot **Central de Atendimento**

Criado por **Gaby.silva**



## Funções
- Painel com embed parecido com o da imagem
- Select menu com opções:
  - Denúncia
  - Doação
  - Suporte
  - ADV
- Cria canal privado automaticamente
- Botão para fechar ticket
- Botão para assumir ticket
- Mensagem inicial dentro do ticket
- Log de abertura e fechamento
- Evita ticket duplicado por usuário/categoria

## Instalação
```bash
npm install
```

Copie `.env.example` para `.env` e preencha.

## Iniciar
```bash
npm start
```

## Comando
Use no servidor:
```bash
/painel-ticket
```

## Permissões do bot
O bot precisa conseguir:
- Ver canais
- Enviar mensagens
- Gerenciar canais
- Gerenciar cargos
- Ler histórico de mensagens

Para criar canais com permissões, o bot precisa de permissões adequadas no servidor. Veja a documentação do Discord sobre permissões e overwrites. citeturn1search2turn1search5
